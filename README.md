# 🛒 E-Commerce MySQL Project

## 📌 Advanced Query Generation for E-commerce Insights
This project focuses on designing a relational database for an e-commerce platform and using advanced SQL queries to extract actionable business insights.

The goal is to simulate a real-world e-commerce analytics environment where SQL is used to analyze customers, products, orders, and operational data to support business decision-making.

---

# 📊 Project Objectives

- Design a relational database schema for an e-commerce platform
- Implement SQL queries to generate meaningful business insights
- Analyze customer behavior and purchasing patterns
- Evaluate product performance and inventory levels
- Generate operational insights related to orders, shipments, and discounts

---

# 🗄️ Database Design

The database consists of multiple related tables structured using primary keys, foreign keys, and constraints to maintain data integrity.

### Tables Included

| Table Name | Description |
|------------|-------------|
| **Customers** | Stores customer information and purchase history |
| **Orders** | Contains order details such as order date, amount, and order status |
| **Products** | Stores product information including price, stock levels, and reviews |
| **Shipments** | Tracks order delivery status and shipment timelines |
| **Discounts** | Contains promotional offers and discount usage details |
